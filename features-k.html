TRACKTOOL

Features

Everything this powerful music management tool can do for your library

🎵 Loading & Playback

  • Load folder, single file, or add folder
  • Drag and drop support — works in Chrome, Edge, Firefox
  • Fast parallel batch loading (10 tracks at a time)
  • Progressive track display as batches complete
  • Track playback with stop and auto-advance
  • Pause/resume without restarting track
  • Skip silence between tracks
  • Scrub through tracks with seek control
  • Volume control
  • Played markers with count and time update
  • Total time counter showing track count and duration

🔍 Fingerprinting & Metadata

  • AcoustID audio fingerprinting — single track
  • AcoustID batch Fix Metadata — tested on 53 tracks
  • MusicBrainz data lookup for title, artist, album, genre, and year
  • Edit panel (pencil icon) for manual metadata editing
  • Editable fields include title, artist, album, genre, year, composer, comments, and album art
  • Apply changes for the current session, or save permanently to disk
  • Save to disk — overwrites original file
  • Save to disk confirmation dialog
  • Note icon on track row with click to open edit panel

⚖ Track Leveling

  • Auto-normalizes playback volume to ~-14 LUFS broadcast standard
  • Consistent levels across tracks from different sources
  • Applied live during playback — original files are never modified
  • No pre-processing or batch analysis step required

📋 Display & Interface

  • Virtual scroll for smooth large library browsing
  • Sticky header stays visible while scrolling
  • Filename toggle (shown/hidden) in amber
  • Duplicate detection — complete rewrite, duration excluded
  • Clear Played button
  • Find playing track
  • Dark interface with animated splash screen on startup

🛡 Reliability & Stability

  • Try/catch error handling throughout
  • Buffer validation and global error logger
  • Reduced outro silence — smooth track transitions
  • Help page covering all features
  • About modal
  • README included with installer
  • Auto-updater — checks for updates every 24 hoursComing in v2

⏱️ Professional Show Timer

  • Broadcast timer for radio shows, podcasts, and DJ sets
  • Select show length: 1, 2, 3, or 4 hours
  • 10-second countdown before timer starts (amber-to-red fade)
  • Counts down from selected duration to zero
  • Color-coded warning zones: green (first 80%), amber (80-90%), red (final 10%)
  • Flashes red for 5 seconds when it hits zero
  • Overtime mode — keeps counting as negative time with "OVER" label until RESET
  • Runs independently of track playback

🎚 Organization & Search

  • Real-time search — filter by title, artist, album, or filename
  • Column sorting — click header to sort, click again to reverse
  • Drag-to-reorder tracks with the ⠿ handle
  • Web search button (⌕) to look up artist or track online
  • Duplicate detection by title + artist
  • Batch remove duplicates with review
  • Stats panel — total tracks, playtime, top artists, albums, and genres
  • Click any stat to filter the list by that artist, album, or genre
  • Export track list to .txt or .csv

How It Works

1

LOAD YOUR MUSIC

Drop a folder or individual files straight into TrackTool. It loads fast — even large libraries appear track by track as they come in.

2

FIX YOUR METADATA

Use AcoustID fingerprinting to automatically identify tracks and fill in missing or wrong metadata. Run it on one track or your whole library at once.

3

SAVE TO DISK

Happy with the result? Hit save and TrackTool writes the corrected tags directly back to your original files. No exports. No copies. Clean and simple.
